# 마감재 실물 사진 폴더

시뮬레이터 스와치에 표시할 마감재 사진을 이 폴더에 넣습니다.

## 올리는 방법 (GitHub 웹에서)

1. 이 폴더 화면에서 **Add file → Upload files** 클릭
2. 사진을 드래그해서 올리고 **Commit changes** 클릭
3. 구글 시트의 `image` 열에 파일명을 적기 — 예: `oak.jpg`

## 사진 규격 권장

- 가로 600px 내외, JPG
- 300KB 이하 (사이트 로딩 속도)
- 텍스처가 잘 보이는 근접 촬영 컷, 정면·균일 조명
- 파일명은 영문 소문자 (예: `oak.jpg`, `walnut-herringbone.jpg`)

`image`가 비어 있는 재료는 시트의 `color` 색상으로 표시되므로, 사진은 준비되는 대로 하나씩 채우면 됩니다.
