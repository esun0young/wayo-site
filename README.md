# Wayo 인테리어 스튜디오 웹사이트

프레임워크·빌드 도구 없는 순수 정적 사이트입니다. 파일을 서버에 올리기만 하면 동작합니다.

## 파일 구성

| 파일 | 내용 |
|---|---|
| `index.html` | 홈 (스튜디오 소개 · 아카이브 · 문의) |
| `simulator.html` | 리모델링 시뮬레이터 (마감재 선택 → 3D 뷰 + 예상 공사비) |
| `grandhill-case-study.html` | 그랜드힐 컨벤션 케이스 스터디 |
| `data/materials.csv` | 마감재 가격표 양식 — 구글 시트로 가져오기용 |

## 로컬에서 열어보기

파일을 더블클릭해도 되지만, 구글 시트 연동까지 확인하려면 간단한 서버로 여는 것을 권장합니다:

```bash
cd 06_홈페이지
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 열기
```

## 마감재 가격표를 구글 시트로 운영하기

관리자가 코드를 건드리지 않고 **시트에서 이름·가격·색상을 수정**하면 사이트에 반영됩니다.

### 최초 설정 (한 번만)

1. [Google Sheets](https://sheets.google.com)에서 새 스프레드시트 생성
2. **파일 → 가져오기 → 업로드**에서 `data/materials.csv` 선택 (현재 시트 바꾸기)
3. **파일 → 공유 → 웹에 게시** → 형식을 **쉼표로 구분된 값(.csv)** 으로 선택 → 게시
4. 생성된 URL(`https://docs.google.com/spreadsheets/d/e/.../pub?output=csv`)을 복사
5. `simulator.html`을 열어 상단의 `SHEET_CSV_URL = ''` 따옴표 안에 붙여넣기

### 시트 열 설명

| 열 | 의미 |
|---|---|
| `category` | `floor`(바닥) `wall`(벽 방식) `paint`(도장 색) `paper`(도배 색) `artwall`(아트월) `tile`(타일) `molding`(몰딩) `light`(조명) `config`(공통 설정) |
| `key` | 영문 식별자. 기존 key는 수정하면 덮어쓰고, 새 key를 추가하면 옵션이 새로 생깁니다 |
| `label` | 화면에 표시되는 한글 이름 |
| `price` | 가격(만원). `config` 행에서는 기본 공사비·배율 값 |
| `color` | 스와치/3D 뷰 색상 (예: `#C99A66`) |
| `option` | `floor`: `wood`면 나뭇결 표현 / `tile`: 타일 한 변 크기(m) |
| `image` | 마감재 실물 사진 파일명 (예: `oak.jpg`). 비어 있으면 `color`로 표시 |

### 마감재 실물 사진 올리기

1. GitHub 저장소의 `assets/materials` 폴더에서 **Add file → Upload files**로 사진 업로드 (규격: 가로 600px 내외 JPG, 300KB 이하)
2. 구글 시트의 `image` 열에 파일명 입력 — 예: `oak.jpg`
3. 사진이 있으면 스와치에 실물 사진이, 없으면 `color` 색상이 표시됩니다. **3D 뷰어는 항상 `color`로 칠해지므로** 사진을 올려도 대표색을 함께 적어주세요.

### 운영 시 알아둘 것

- 시트 수정 후 사이트 반영까지 **몇 분 정도 걸릴 수 있습니다** (구글의 게시 캐시).
- 시트를 불러오지 못하면(오프라인 등) `simulator.html`에 내장된 기본 가격표로 동작합니다. 시트를 바꿀 때 내장 기본값도 가끔 맞춰주면 안전합니다.
- `wall`의 세 방식(paint/paper/mix)과 `artwall`의 slat/stone 무늬는 코드와 연결되어 있어 **key 이름은 바꾸지 말고** label·price만 수정하세요.

## 도면 업로드 기능 (시뮬레이터)

- 뷰어 우측 상단 **"내 도면 올리기"** 버튼으로 이미지(JPG/PNG)를 올리면 뷰어에 표시됩니다.
- 참고용 표시 기능입니다 — 파일은 **브라우저 안에서만 사용되고 어디에도 전송·저장되지 않으며**, 새로고침하면 사라집니다.
- ＋/−로 확대, 드래그로 이동이 가능합니다.
- **영역 그리기 ✎**: 확대/축소와 별개로, 도면 위에 클릭으로 방 모서리를 표시해 다각형을 만들고 "바닥" 또는 "타일"로 태그하면, 그 영역이 현재 선택된 바닥/타일 마감재 색으로 실시간 채워집니다. 마감재를 바꾸면 칠해진 영역 색도 함께 바뀝니다.
  - 실측 정보가 없어 **면적 기반 견적 계산에는 연결되어 있지 않습니다** — 어디까지나 시각적 미리보기입니다. 하단 예상 공사비는 예시 평면 기준 그대로입니다.

## 알려진 한계

- 시뮬레이터의 "상담 신청" 폼은 데모입니다 — 실제로 이메일이 전송되지 않고 접수 안내만 표시됩니다. 실제 접수가 필요하면 Formspree 같은 폼 서비스 연결이 필요합니다.
- 업로드한 도면 위에 그린 영역은 실제 치수와 무관합니다(실측 배율을 모름). 정확한 면적 견적이 필요해지면 "기준선 실측 입력(캘리브레이션)" 기능 추가를 고려할 수 있습니다.
